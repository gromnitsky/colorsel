Renders X11-like mappings of color names to RGB values. Historically, the famous rgb.txt file has been a primary example of such mappings: 

$ head -2 /usr/share/X11/rgb.txt
255 250 250             snow
248 248 255             ghost white

Clicking on the Color table header attempts to sort the table rows by hue, placing achromatic (or nearly achromatic) colors at the end.

'Filter by' uses a substring match (requires pressing Enter or clicking the Submit button).

'Find most probable names' uses CIE color difference.

History

In the beginning there was xcolorsel program:

It hasn't been updated since 1994, and its author says in the README that he consider[s] this tool as finished. No further extensions, bug fixes are planned.

In 2014 I was leaning a new-wave web framework called Angular.js, along with a hot syntactic-sugar language called CoffeeScript. Around that time, the original xcolorsel started producing core dumps, and instead of fixing its archaic code—which was using a heroic Xaw3d widget library—I wrote a simple 'Angular app' to visualise X11 rgb.txt type files, and named it ColorSel.

Angular.js has been deprecated since 2018, if not earlier. Recently, I wanted to add another color list to ColorSel, but then decided to rewrite the app using a no-framework, no-build-step approach.

Bugs

  • Sorting by hue is suboptimal.
  • When in the 'Find most probable names' mode, clicking on the 'Color' table header should not be allowed, for it does nothing.
  • Scrolling position between 'modes' is not retained.
  • No support for pre-2025 browsers. ☕
